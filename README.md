## 📚 About

This repository contains my solutions for the **C++ Modules (00–04)** from the 42 School curriculum.

These modules introduce **Object-Oriented Programming in C++**, covering fundamental to intermediate concepts through hands-on exercises.

---

## 🧩 Modules Overview

### 🔹 Module 00 — Basics of C++

- Namespaces  
- Classes & Member Functions  
- std::cout / std::cin  
- Static variables  

> Introduction to C++ syntax and basic class structure.

---

### 🔹 Module 01 — Memory & References

- Memory allocation (`new` / `delete`)  
- References  
- Pointers vs References  
- File streams  

> Understanding how memory works in C++ and how to manage it properly.

---

### 🔹 Module 02 — Ad-hoc Polymorphism

- Canonical form (Orthodox Canonical Class Form)  
- Operator overloading  
- Fixed-point numbers  

> Deep dive into operator overloading and class design.

---

### 🔹 Module 03 — Inheritance

- Inheritance  
- Constructors & destructors  
- Base vs derived classes  
- Method overriding  

> Introduction to inheritance and class hierarchies.

---

### 🔹 Module 04 — Subtype Polymorphism

- Polymorphism  
- Virtual functions  
- Abstract classes  
- Interfaces  

> Focus on dynamic polymorphism and proper OOP design.

---

## ⚙️ Compilation

Each module contains its own Makefile.

To compile:
```bash
make
```
To clean:
```bash
clean
```
